About Veridect

Technology stack & methodology

NLP
Natural language processing
Sentence-level analysis using regex-based pattern matching for propaganda tactics, sentiment bias detection via keyword scoring, and linguistic manipulation heuristics. Production version would use fine-tuned BERT/RoBERTa.
spaCyNLTKHuggingFaceBERT
OSINT
Open source intelligence
Domain credibility scoring using a curated whitelist of known reliable sources, detection of suspicious naming patterns, TLD analysis, domain age simulation, and registrar country risk profiling.
WhoisGDELTNewsAPIShodan
ML
Scoring & fusion
Weighted multi-signal fusion model combining NLP signals (40%), source credibility (25%), and cross-source agreement (10%). Final risk classification into 4 tiers. Explainability via per-signal breakdown.
XGBoostSHAPscikit-learnLIME
API
Backend & deployment
Lightweight Flask REST API with JSON responses. All analysis runs server-side. Designed for Docker containerisation. Frontend is pure HTML/CSS/JS — no framework dependencies.
FlaskPythonDockerREST API
